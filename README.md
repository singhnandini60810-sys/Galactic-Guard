Rocket Defender
A retro 90s-style arcade rocket shooting game built using pure HTML, CSS, and JavaScript.

Game Overview
Rocket Defender is a browser-based space shooter where players control a rocket to destroy falling spaceships. The game includes a life system, power-ups, pause functionality, and high score tracking using browser storage.

Controls
Key	Action
Enter	Start Game
Shift	Pause / Resume
Left Arrow	Move Left
Right Arrow	Move Right
Up Arrow	Shoot
Space	Activate Boost (15 seconds)
Down Arrow	Activate Shield (15 seconds)

Game Rules
The player starts with 3 lives.
A life is lost only if a spaceship reaches the bottom of the screen.
The shield prevents life loss for 15 seconds.
Boost increases enemy falling speed for 15 seconds.
The game ends when all 3 lives are lost.
High score is saved using LocalStorage.

Features
Smooth enemy spawning
Collision detection system
Shield and Boost power-ups
High score persistence
Clean restart system
No external libraries
Technologies Used
HTML5
CSS3
Vanilla JavaScript
LocalStorage API

How to Run
Clone or download the repository.
Open index.html in any modern browser.
Press Enter to start playing.
